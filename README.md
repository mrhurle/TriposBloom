Contains code associated with the following paper: Bloom dynamics of Tripos muelleri in the Gulf of Maine during summer 2023 revealed through integrated biological and environmental observations

Hurley, M.R., S. Shellito, R. Letscher, C. Hunt, J.L. Pinckney, D. Vandemark, and E.L. Harvey. (In Review).

Raw sequence data are available in NCBI SRA under BioProject ID PRJNA1431471 

This GitHub account is the original author of the attached code. Code is also available/published on https://github.com/Harvey-Lab-at-UNH/TriposBloom
